=============================
 how to use 1-my_list.py
=============================

    Using ``MyList``
-----------------------------

Firstly import add_integer from 0-add_integer

	>>> MyList = __import__('1-my_list').MyList

Now you can use it:

	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

	negative number
	>>> MyList.print_sorted([1, 2, -3, 4, -4, 1])
	[-4, -3, 1, 1, 2, 4]
	
	empty list
	>>> MyList.print_sorted([])
	[]
	
	is instance
	>>> isinstance(MyList([]), list)
	True
	
	repeted elements
	>>> MyList.print_sorted([1, 1, 3, 2, 1, 0])
	[0, 1, 1, 1, 2, 3]
